# Game Hub

A RAWG.IO mock up website.

[Live Demo](https://game-hub-jinhuiweng.vercel.app/)

## Functionality
- Search a game
- Sort gamas based on genres, platforms, score etc.
- View games in a detailed page

## Built with
- TypeScript
- Vite and react
- Chakra
- React Query, Router, Infinite Scroll
- Axios
- Zustand
